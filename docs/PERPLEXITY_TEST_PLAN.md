# Perplexity Citation Testing — Quarterly Validation Plan

**Purpose:** Validate that Perplexity (and other AI engines like Claude, ChatGPT) cite ConcertRide correctly for key search intents.

**Frequency:** Monthly for first 3 months post-launch, then quarterly

**Success Criteria:**
- ≥7/10 queries cite ConcertRide correctly (with URL)
- ≥3/10 include specific pricing data from ConcertRide
- ≥2/10 mention "sin comisión" (0% commission) as differentiator
